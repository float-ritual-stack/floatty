# Provenance and Links

## Core rule

No durable artifact without a backlink into the outline.

## external_ref examples

- PR
- issue
- file path
- URL
- generated artifact
- dataset
- export

## Purpose

Transforms the outline from:
scratchpad -> index of reality

## Implication

External execution work should point back into outline structure.
The outline does not need to contain the whole artifact.
It needs to remain the map of how that artifact came to exist.
