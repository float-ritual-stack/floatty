## Target Audience
- Developers who need a high-performance terminal with integrated AI-driven context capture.

## Goals
- Provide a seamless terminal experience with high rendering performance using WebGL.
- Enable efficient context gathering and parsing from terminal output for AI workflows.
- Integrate a robust note-taking and outlining system directly into the terminal workflow.

## Core Features
- **High-Performance Rendering:** WebGL-based terminal rendering with support for ligatures and Unicode 11.
- **Consciousness Siphon System:** Monitoring logs for `ctx::` markers, storing them in SQLite, and parsing them via local LLMs (Ollama).
- **Recursive Split-Pane Layout:** A flexible workspace organization system with split panes and tab management.
- **Hybrid Note-Taking/Outliner:** Integrated system for organizing thoughts and notes alongside terminal activity, with future support for agent integration.
